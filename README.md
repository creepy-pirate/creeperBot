# creeperBOT

Twitter bot that retweets tweets based on the Hashtags provided.

## Installation

+ Install [Node.js](http://nodejs.org/)
+ Clone this Repo
```bash
git clone https://github.com/creepy-pirate/creeperBot.git
```

## Usage

```bash
node bot.js
```
